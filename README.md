# branch

>## branch 활용
- branch : 독립적인 작업공간을 만들고 관리

>### branch 주요 명령어
```bash
git branch [브랜치이름] : 브랜치생성
git checkout : 브랜치이동
git merge : 브랜치병합
git branch -d : 브랜치삭제
```
